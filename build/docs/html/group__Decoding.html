Detailed Description

Function objects that encode ranges of bytes.

Decoding function objects accept a range of integral types. Input characters are narrowed to the range [0, 255). You can optionally pass a projection that converts the range value type to 'char'. The default projection uses static_cast.

Multiple overloads are available, you can use pairs of iterators or input ranges or iterator + size.
