\doxysubsection{Detailed Description}
Function objects that encode ranges of bytes. 

Decoding function objects accept a range of integral types. Input characters are narrowed to the range \mbox{[}0, 255). You can optionally pass a projection that converts the range value type to \textquotesingle{}char\textquotesingle{}. The default projection uses static\+\_\+cast.

Multiple overloads are available, you can use pairs of iterators or input ranges or iterator + size.
